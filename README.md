##  Installation (Local Developer Mode)

Since this extension is optimized for lone operators and early access builders, you can load it directly into your Chromium-based browser (Chrome, Brave, Edge, Arc) using Developer Mode. 

Follow these 3 simple steps:

### Step 1: Download & Unzip
1. Clone this repository or download the source code as a ZIP file to your local machine.
2. Extract the ZIP file into a permanent directory (e.g., `Documents/ProductiveOS`). 
   * *Note: Do not delete or move this folder after installation, or the extension will break.*

### Step 2: Enable Developer Mode
1. Open your browser and navigate to the Extensions management page:
   * **Chrome/Brave/Edge:** Type `chrome://extensions` in your URL bar and hit Enter.
   * **Arc Browser:** Go to Settings > Extensions.
2. In the top-right corner of the Extensions page, locate the toggle switch labeled **"Developer mode"** (Chế độ nhà phát triển) and turn it **ON**.

### Step 3: Load the Extension
1. Look at the top-left corner of the page and click the **"Load unpacked"** (Tải tiện ích đã giải nén) button.
2. A file picker dialog will appear. Select the **root folder** of the extracted project (the directory containing the `manifest.json` file).
3. Click **Open** / **Select Folder**.

** Productive OS is now live.** Pin it to your toolbar and open a new tab to enter your clean environment.
